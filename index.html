A propos du Meetup

Le web évolue, les tendantes évoluent. Le web a connu une popularité immense qui ne cesse d'accroître au fil du temps, et des technologies naissent de plus en plus. Javascript se presente aujourd'hui comme la technologie au coeur du web moderne: le WEB 2.0, fini le farfeulu code javascript qui etait écrit dans les balises script en bas de page. Un vrai ecosysteme de developpement s'est batti autour de Javascript (js) avec notemment l'arrivee de NodeJS qui permet desormais de developper cote serveur avec Javascript et qui comme boosté l'attention de toute le monde au sujet de Javascript.

Ils soutiennent le projet

Connaître javascript c'est bien, mais connaître les bonnes pratiques d'entreprise c'est encore mieux. C'est dans cette logique que nous avons organisé ce Meetup qui a pour but de vous introduire dans le javascript dit moderne et faciliter votre entrée dans le monde du web.
Nous parlerons surtout de AngularJS, de la mise sur pied d'une application basique a la connexion à une REST-API, l'application exemple sera "un Store de vente de Smartphones", il sera question de simuler toutes les interactions avec un utilisateur réel

Programme



HTML5-CSS3-Bootstrap, L'ère du Responsive (20 Minutes)

HTML5/CSS3 ont apporté des nouveautés au web, aujourd'hui il est facile de creer des sites web responsive Design qui s'adapte à tous les terminaux, ces tecnhnologies sont arrivées pil au moment ou le monde de la technologie connait une évolution grandiose chacun désormais depuis son smartphone veut lire les informations sur son site d'actualités préfére en gardant le meme confort et la meme expérience

jQUery,Dojo et compagnie (20 Minutes)

Jquery, Dojo, Prototype sont les Libraries qui ont longtemps [et qui continuent de ] dominer le web, Le DOM est alors manipule grâce aux fonctions que Jquery offre. On rend ainsi alors facilement les pages web dynamiques. Aujourd'hui encore beaucoup d'entreprises les utilisent pour raison de leur maturité et de la communauté qui est derriere ces librairies. Cependant quand on parle de Web moderne on parle aussi te technologies modernes et ces librairies ne sont plus en mesure de repondre correctement aux attentes des developpeurs d'aujourd'hui, avec la montee en puissance des Single Pqge Application tout devient complexe à gérer avec ces outils

AngularJS le web moderne (2 Heures)

AngularJS est un framework créé et maintenu par une équie de Google. Il fait partie des outils qui ont revolutionés le développement web. Les Single Page Applications qui sont la tendance actuelle du developpement web ont vu leur développement être facilité. AngularJS impose au developpeur l'utilisation du Design pattern MVC/MVVM qui est le design de Prédilection des applications de toutes sortes. AngularJS est donc devenu un acteur important dans le web d'aujourd'hui et derrière lui s'est battie une vraie communaute de developpeurs

Installez Git

Git est un Systeme distribue de gestion de versions. C'est un des outils preferes des developpeurs et nous n'en faisons pas l'exception. Pendant le Meetup, des codes exemples seront stockes sur un repertoire Git prepare pour l'occasion, il sera donc question pendant le Meetup que tous les participants aient une version de Git qui marche et qu'ils soient en mesure de telecharger et tester les exemples.

Installez NodeJS sur votre ordinateur

NodeJS est la plateforme qui a permis de faire du Javascript cote serveur, pouvant completement remplacer Java/Python/PHP ou C#. C'est une technologie qui est devenu incontournable pour le developpement d'applications web modernes. Nous utiliserons NodeJs en ligne de commande pour installer les outils necessaires au developpement avec AngularJS

Pour les utilisateurs Windows rendez-vous sur le site de NodeJS et telechargez l'Executable, executez le et vous avez NodeJS
Pour les utiisateurs de Linux utilisez tout simplement la commande suivante pour avoir NodeJs installe. 

sudo apt-get install nodejs
